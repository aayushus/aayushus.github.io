⚖️
Humans stay in control — agents stop, ask, and wait
Agents never merge without human approval. Branch protections apply equally to agents and engineers.
If acceptance criteria are missing or ambiguous, Coder posts a Jira comment asking for clarification and stops.
No auto-merge for auth, payments, or database changes — Sentinel flags these for mandatory human review.
P2/P3 bug fixes run autonomously through Critic auto-review — but P1 fix PRs always require human approval before they merge.
Override any running agent from Teams at any time. Send EXIT to end a session immediately.
